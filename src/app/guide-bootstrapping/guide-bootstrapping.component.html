Boot Strapping

NgModule用於描述AppModule應用的各個部如何組織在一起

  • declarations -- 應用所擁有的元件
  • imports -- 匯入BrowserModule以獲取瀏覽器的服務
  • providers -- 各種服務提供者
  • bootstrap -- 根元件，Angular建立它並插入index.html頁面

declarations

declarations陣列只能接受可宣告物件。

declarations陣列只能接受可宣告东件。

透過@NgModule使用指令

  1. 從你編寫的檔案匯出它
  2. 匯入到模組中
  3. @NgModule的declarations陣列中宣告它

imports

告訴Angular該模組想要正常工作，還需要哪些模組

providers

列出所需要的服務

bootstrap

每個元件都有自己的元件樹的根

根元件AppComponent，放在bootstrap陣列
